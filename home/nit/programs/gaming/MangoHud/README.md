# MangoHud Configuration

This directory contains the configuration for the **MangoHud** performance overlay.

## Files

| File | Description |
|---|---|
| `MangoHud.conf` | Overlay configuration that defines which metrics to display (FPS, CPU, GPU, RAM, VRAM, power, etc.), position (top-center), horizontal format and toggle key (F12). |

MangoHud is used to monitor the performance of games and graphics applications in real time, allowing identification of performance issues and verifying that games are running correctly.
