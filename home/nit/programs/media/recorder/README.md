# Recorder Script

This directory contains the screen recording script.

## Files

| File | Description |
|---|---|
| `recorder.sh` | Bash script that wraps **wf-recorder** for recording the screen. Simplifies the recording process by providing a simpler interface than using wf-recorder directly with all its options. |

wf-recorder is chosen for being a native Wayland recording tool that works well with Niri and other Wayland compositors.
